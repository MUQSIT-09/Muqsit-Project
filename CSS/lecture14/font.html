FallBacks in Font-Family

By default there will be one after other font family so that for any reason if the first one is not available then the next one is taken. Ex:font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;

If you want to take your own font-family which is not available in browser then you can use your own family from google font(External Font)

Browsers have all the font-family that is known as web safe font, which is already available in browser universally

Select your font and then click on get the code then copy the link tag and paste in head tag and then copy the font-family code and paste in stylesheet.

Colors In CSS:

How can we add colors ,In How Many Ways

1.Named Colors

Add Directly in background-color,color by giving the color name

2.RGB Values

1.You Can give any number inside rgb() and then by hovering choose/set the color shade you want

2.You can also give the alphanumeric value inside the rgba(), where the alpha value decides the transparency of the color

3.HexaDecimal Values

Go to browser and search for HexaDecimal color picker you can take the value of the color and apply inside stylesheet

4.HSL = Hue Saturation and Lightness

    1.Hue is a degree on the color wheel from 0 to 360. 0 (or 360) is red, 120 is green, 240 is blue.
    2.Saturation can be described as the intensity of a color. It is a percentage value from 0% to 100%.
      100% is full color, no shades of gray.
      50% is 50% gray, but you can still see the color.
      0% is completely gray; you can no longer see the color.
    3.The lightness of a color can be described as how much light you want to give the color, 
      where 0% means no light (dark), 50% means 50% light (neither dark nor light), and 100% means full light.

HSLA Color Values:

HSLA color values are an extension of HSL color values, with an Alpha channel - which specifies the opacity for a color.
