Transition

Switching from one state to another state of styling

There is a single switch from source to destination but in animation there is a series/set of complex movement

How to Use:

Define the starting state (e.g., background color is blue).
Apply a transition rule (e.g., background color changes over 1 second).
Change the property on an event (e.g., when you hover over the element, it turns red).

Pesudo class

A pseudo-class in CSS is used to style an element based on its state or position, without needing to add extra classes or IDs in HTML.
