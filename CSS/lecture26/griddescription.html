What is Grid

CSS Grid is a layout system that allows you to create flexible, two-dimensional web layouts using rows and columns.
It makes it easy to align and arrange elements without using floats or positioning tricks.

Powerfull Layouts are made with the combination of grid and flex

Types Of Grid

1. Block (display: block;)

Takes up the full width of its parent container.
Starts on a new line.
Example: div, p, h1

2. Inline-Block (display: inline-block;)

Behaves like an inline element (stays in line with other elements).
Respects width and height properties (unlike inline).
Example: span style="display: inline-block;">Text span>

3. Inline-Grid (display: inline-grid;)

Behaves like inline-block but uses CSS Grid layout inside. The element stays in line but allows child elements to be arranged using grid properties.


CSS Grid Properties:

Grid Container Properties

display

Defines a grid container (grid or inline-grid).

grid-template-columns

Specifies column sizes and numbers.

grid-template-rows

Specifies row sizes and numbers.

grid-template-areas

Defines named grid areas.

grid-template

Shorthand for grid-template-rows, grid-template-columns, and grid-template-areas.

grid-auto-columns

Defines size of auto-generated columns.

grid-auto-rows

Defines size of auto-generated rows.

grid-auto-flow

Controls automatic placement (row, column, dense).

grid-row-gap

Defines vertical space between rows.

grid-column-gap

Defines horizontal space between columns

Grid Item Properties

grid-column-start

Defines where an item starts in columns.

grid-column-end

Defines where an item ends in columns.

grid-column

Shorthand for grid-column-start and grid-column-end.

grid-row-start

Defines where an item starts in rows.

grid-row-end

Defines where an item ends in rows.

grid-row

Shorthand for grid-row-start and grid-row-end.

grid-area

Assigns a grid item to a named grid area.

FR-Fractuinal Unit

A flexible unit used in CSS Grid to distribute space proportionally. For example, grid-template-columns: 2fr 1fr; means the first column gets twice the space of the second column.

Alignment Properties

justify-items

Aligns items horizontally (start, end, center, stretch).

align-items

Aligns items vertically (start, end, center, stretch).

place-items

Shorthand for align-items and justify-items.

justify-self

Aligns an individual grid item horizontally.

align-self

Aligns an individual grid item vertically.

place-self

Shorthand for align-self and justify-self.

justify-content

Aligns the entire grid horizontally (start, end, center, space-between, etc.).

align-content

Aligns the entire grid vertically.

place-content

Shorthand for align-content and justify-content.
