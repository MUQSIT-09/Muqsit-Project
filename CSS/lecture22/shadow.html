box-shadow

box-shadow: 5px 5px 10px gray;

First value: Horizontal offset (X-axis), If you give in negative then it will come in opposite side. Second value: Vertical offset (Y-axis) Third value (optional): Blur radius (big number = blur will become more bigger,color will get more lighter) there is no negative radius for blur. Fourth value (optional): Spread (The value specifies that how much the shadow should spread) Last: Color

Custom Variable

It will be applicable within the braces only that's if we want to use again and again we can declare in root


Define a Variable:

Declaring the root: to use anywhere you want

:root { --primary-color: #3498db; --font-size: 16px; }


Use the Variable:

body { background-color: var(--primary-color); font-size: var(--font-size); }

Update Variables Dynamically:

body.dark-mode { --primary-color: #1a1a1a; }
