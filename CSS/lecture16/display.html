Block Element:

        
    1. New Line
    2. Width Takes From Extreme Left To Right.
    3. Custom Width/Height    -> Respect
    4. Custom Margin/Padding  -> Respect

Box-sizing : boeder-box

The actual size of the box doesn't gets impacted/it is retain, if it is not used then the actual size is not retain and gets expand

Inline Element

        
    1.Same Line(If there is no space/width it goes to next line)
    2.Width is the length of the content.
    3.Can't Apply Custom width and height.
    4.Can Apply Margin,Padding only horizontally, but not vertically.

Can Convert to each other Vice-Versa

Display:Block , Display:inline

If You want mixture of Both inline+block

Then you can use Display:inline-block
