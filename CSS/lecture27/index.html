auto-fill

Fills the row with as many columns as possible, even if they are empty.
If extra space is available, it maintains the grid structure by keeping "phantom" empty columns.

auto-fit

Tries to fit as many columns as possible, collapsing extra empty tracks.
If extra space is available, it expands the existing items to fill the space.
