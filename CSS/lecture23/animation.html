Animations In CSS

Allow elements on a webpage to move or change styles over time.

br

1. @keyframes (Defines Animation)

Specifies the animation steps (from → to or percentages).
@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } }

2. animation-name

Links an element to a @keyframes animation.
div { animation-name: fadeIn; }

3. animation-duration

Sets how long the animation lasts (e.g., 2s, 500ms).
div { animation-duration: 2s; }

4. animation-timing-function

Controls the speed curve (ease, linear, ease-in, ease-out, ease-in-out).
div { animation-timing-function: ease-in-out; }

It also ahve Cubic Bezier

Cubic Bezier (cubic-bezier(x1, y1, x2, y2))
The animation-timing-function property defines how an animation progresses over time. It controls the speed curve of the animation.
A Cubic Bezier curve is a custom speed curve used to control acceleration and deceleration.
It takes four numeric values between 0 and 1, which define the curve's shape.

5. animation-delay

Delays the animation start (2s, 0.5s).
div { animation-delay: 1s; }

6. animation-iteration-count

Sets how many times the animation repeats (1, infinite).
animation-iteration-count: infinite;

7. animation-direction

Controls the direction (normal, reverse, alternate, alternate-reverse).
animation-direction: alternate;

8. animation-fill-mode

Defines styles before/after animation (none, forwards, backwards, both).
animation-fill-mode: forwards; in the case of both = both applies the styles before the animation starts (from backwards) and after it ends (from forwards).

9. animation-play-state

Pauses (paused) or plays (running) an animation.
animation-play-state: paused;

Shorthand Property (animation)

animation: fadeIn 2s ease-in-out 1s infinite alternate;
fadeIn → animation name 2s → duration ease-in-out → timing function 1s → delay infinite → repeats forever alternate → reverses direction every cycle
