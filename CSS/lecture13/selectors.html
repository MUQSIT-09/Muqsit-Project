Selectors

Element Selector

You can select the Elementa and apply styling in it Ex:h1,p,section,article we can select and aplly style

Id Selector

You can assign an id for an Element and use that id for styling

You should use '#' before the id to access it for styling

Id is unique for an Element

Class Selector

You can assign an calss for many Element and use that class for styling

You should use '.' before the class to access it for styling

Class is not unique for an Element, you can group multiple Element for an single class

Group Selectors

we can keep ',' among the Elements and apply the css styling by grouping them together.
