Box Model In CSS:

Box model states that any content rendering/display in the browser, that can be displayed in rectangular box shape only.

Box Model Have 4 Attributes

1.Content

The main stuff/content inside the element (like text, images, etc.).

2.Padding

The space/layer between the content and the border is your Padding, Padding can be zero also

3.Border

The line around the padding (optional).

4.Margin

The space/Gap between the two boxes is your Margin,  (Or)
The space outside the border that separates the element from other elements

By Defaukt there will be some padding.margin

Then you can use the uNiversal selector and set the padding and margin to zero

Box-Sizing

Is a CSS property that defines how the total width and height of an element are calculated. It determines whether padding and borders are included in the element's dimensions or if they are added outside the specified width and height.

content-box

The width and height include only the content of the element. Padding and borders are added outside the defined width and height, increasing the overall size of the element.

border-box:(Best Practice)

The width and height include content, padding, and borders. Padding and borders do not increase the element's size beyond the specified dimensions
