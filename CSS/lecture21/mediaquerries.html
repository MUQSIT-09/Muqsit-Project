MediaQuerry

Helps make websites responsive. It allows a webpage to change its style based on the screen size, resolution, or device type (like mobile, tablet, or desktop).

min-width

The styles apply only when the screen width is greater than or equal to the given value.
Used for progressive enhancement (styles apply for larger screens).

max-width

The styles apply only when the screen width is less than or equal to the given value.
Used for mobile-first designs (styles apply for smaller screens).

Both Together In A Range

You can combine both to target a specific range of screen sizes.
Ex:@media (min-width: 601px) and (max-width: 1024px)
