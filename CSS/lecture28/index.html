3D Transformations in CSS

Rotat3

rotateX

Rotates the element around the X-axis.

rotateY

Rotates the element around the Y-axis.

rotateZ

Rotates the element around the Z-axis.

Translate (Move)

translateX(value)

Moves the element along the X-axis.

translateY(value)

Moves the element along the Y-axis.

translateZ(value)

Moves the element along the Z-axis.

Scale (Resize)

scaleX(value)

Scales the element along the X-axis.

scaleY(value)

Scales the element along the Y-axis.

scaleZ(value)

Scales the element along the Z-axis.

Short-Hand

transform: rotateX(30deg) translateZ(50px) scaleY(1.5);

Perspective in 3D Transformations

A sense of depth by determining how far the viewer is from the 3D object.
It makes elements appear closer or farther along the Z-axis.

Basically from how much sitance you are seeing the object
