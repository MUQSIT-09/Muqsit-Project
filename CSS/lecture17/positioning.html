CSS Positioning

What is CSS Positioning?

How to position the elements in the webpage

Types Of Positioning

1.Static

Elements appear in the normal document flow.
No effect from top, right, bottom, or left,zindex.
By Default The position is static.

2.Relative

Positioned relative to its normal position.
Can use top, right, bottom, and left to shift it.
The Image Gap/Space is till maintained.

3.Absolute

If You Want to do overlapping,
Positioned relative to the nearest positioned ancestor (relative, absolute, or fixed). If no ancestor is positioned, it is placed relative to the (viewport). The Image Gap/Space is not maintained.

4.Fixed

Positioned relative to the viewport.
Stays in the same place even when scrolling.
fixed is fixed correspondong the whole window, doesn't matter with any parent it is applicable to entire webpage.

5.Sticky

Acts like relative but becomes fixed when scrolled past a threshold.
Requires a top, bottom, left, or right value.
Sticky shows sticky behaviour upto it's parent container,
But fixed is fixed correspondong the whole window, doesn't matter with any parent it is applicable to entire webpage.
